## Correr los tests
Para correr los tests, se debe ejecutar el siguiente comando en la terminal:
- Ejecutar todos los tests
```bash
sh test.sh
```
- Ejecutar un test en específico
```bash
sh test.sh nombre_del_test
```

o 

```bash
python3 -m test.nombre_del_test
```
